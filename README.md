A machine learning application that predicts whether a news article or headline is **REAL** or **FAKE**. Built using **Python**, **scikit-learn**, and **Streamlit**, this project allows users to paste news text and instantly get a prediction.

## Features

- Classifies full-length articles or headlines as **FAKE** or **REAL**
- User-friendly **web interface** using Streamlit
- Trained on a dataset of real and fake news articles
- Preserves punctuation and formatting for better predictions
